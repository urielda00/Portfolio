# Uriel Dahan - Portfolio

A personal portfolio website built to showcase my projects, skills, and development work in one place.

## Live Demo
[View Portfolio](https://uriel-dahan-portfolio.netlify.app/)

## About the Project

This portfolio was created as a central place to present my work in a clean and accessible way.  
It includes selected featured projects, additional project cards, and links to source code and live demos when available.

## Features

- Modern and responsive UI
- Featured projects section
- Project cards with GitHub and live links
- Clean layout for presenting skills and work
- Built for easy browsing by recruiters and employers

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS

## Notes

Some projects in the portfolio use backend services hosted on Render's free tier, so initial server response may take up to a minute if the service is waking up.
